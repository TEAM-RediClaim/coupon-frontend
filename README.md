# 쿠폰 발급 시스템 (프론트엔드)

쿠폰 발급 시스템의 프론트엔드 프로젝트입니다.

## 기술 스택

- React
- TypeScript
- Axios (HTTP 요청)

## 실행 방법

### 의존성 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm start
```
웹 브라우저에서 http://localhost:3000 으로 접속하여 앱을 사용할 수 있습니다.

## 빌드

```bash
npm run build
```

## 주요 기능

- 사용자 ID와 쿠폰 ID를 입력하여 쿠폰 발급
- 응답 상태에 따른 성공/오류 메시지 표시
